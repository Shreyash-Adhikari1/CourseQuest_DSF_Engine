# Computing Pathway
## BSc (Hons) Software Engineering

---

## Course Overview

This pathway focuses on the design, development, and deployment of software systems using contemporary engineering principles and industry-standard practices. It is the broadest of the three pathways — covering full-stack web development, mobile applications, systems analysis, data science, UX design, software engineering methodology, and security.

Where AI goes deep into intelligent systems and Cybersecurity goes deep into attack and defence, Computing goes wide — producing graduates who can operate across the full software development lifecycle, from requirements gathering and system design through to implementation, testing, and deployment.

The program is structured across three years. Year 1 establishes programming and computing fundamentals. Year 2 introduces full-stack development, software engineering methodology, and data science. Year 3 covers mobile development, web APIs, UX design, security, and a final dissertation project.

---

## What You Will Study — Year by Year

### Year 1 — Programming and Computing Foundations
- **Programming: Concepts and Algorithms** — Python, algorithms, data structures, testing, version control (Git)
- **Mathematical Skills for Computing Professionals** — Propositional logic, sets, functions, relations, number systems, graphs, sorting and packing algorithms
- **Computer Systems** — Database management systems, data modelling, database design principles, hands-on with MySQL and Oracle SQL
- **Programming: Professional Practice** — Algorithm design and complexity, storage technology, statistical analysis, introduction to machine learning tools
- **Working with Data** — Data collection, cleaning, transformation, visualisation, and analysis using SQL, Python, and spreadsheet tools
- **Integrative Project** — Capstone project integrating Year 1 knowledge to solve a real-world problem

### Year 2 — Full-Stack Development and Software Engineering
- **Web Development** — Full-stack web application development, Django, Flask, client-side scripting, databases, networking fundamentals (HTTP, server-client architecture)
- **Advanced Algorithms** — Heaps, graphs, trees, hash tables, greedy algorithms, backtracking, multi-threading, functional programming, space/time complexity analysis
- **System Analysis and Design** — Software development models (Agile, Scrum, Waterfall, Kanban), requirements engineering, BDD, TDD, software architectural patterns, CI/CD pipelines
- **People and Computing** — User interface design, usability principles, computer law, copyright, IP, data protection, professional ethics
- **Data Science** — Data mining, big data, statistical analysis, probability, NoSQL databases, Hadoop, data visualisation (Python, R, MongoDB, Cassandra)
- **Software Engineering** — Requirements engineering, software design patterns, testing and QA, project management, documentation, Agile practices

### Year 3 — Specialization and Applied Development
- **Project Discovery** — Research methodologies, academic research preparation, project planning, literature review, ethics approval
- **Mobile Application Development** — Flutter, Swift, Kotlin, mobile security, authentication, data security, cloud integration, design patterns
- **Web API Development** — RESTful API design and development, web security and authentication, client-side frameworks (React, Angular, Vue), Node.js, MongoDB
- **Dissertation and Project Artifact** — Independent research and development project — a working software product with evaluation and written report
- **Security** — Cryptography, PKI, secure programming, SSL/HTTPS, session management, hashing, multi-factor authentication, blockchain basics, network security
- **User Experience Design** — User-centered design, interface prototyping, Figma, usability evaluation, GUI development

## Career Opportunities

Graduates of this pathway are prepared for roles including:

- **Software Engineer / Developer** — building production software systems across any domain
- **Full-Stack Web Developer** — frontend and backend web application development
- **Mobile Developer** — iOS, Android, or cross-platform mobile application development
- **Backend Developer** — API development, server-side systems, database integration
- **Frontend Developer** — client-side interfaces, user experience implementation
- **Systems Analyst** — analysing requirements and designing software solutions
- **DevOps Engineer** — CI/CD pipelines, deployment automation, infrastructure
- **UX/UI Designer** — user interface and experience design
- **Data Analyst** — data processing, visualisation, and reporting
- **QA Engineer / Software Tester** — quality assurance and test automation
- **IoT Developer** — connected devices and pervasive computing systems
- **IT Support / Systems Administrator** — infrastructure and systems management

## Who This Pathway Suits

This pathway is best suited to students who:

- Enjoy building things — applications, systems, products that users actually interact with
- Want breadth across the software development landscape rather than depth in one specialist area
- Are interested in how software is engineered professionally — not just written, but designed, tested, maintained, and deployed
- Enjoy working across the full stack — from database design through backend logic to user-facing interfaces
- Are interested in user experience — how people interact with software matters as much as how it works
- Want flexibility in their career — this pathway opens the widest range of job roles of the three pathways
- Are drawn to web and mobile development as primary domains

Strong programming fundamentals and genuine interest in systems and how they are built are the primary indicators of fit. This pathway rewards students who are practical builders — people who get satisfaction from shipping working software.

---

## How This Pathway Differs from the Others

Compared to AI, this pathway covers more ground — web, mobile, UX, APIs, software engineering methodology — but goes less deep into mathematics, data science, and intelligent systems. Students who want to eventually work in AI but prefer a broader foundation sometimes choose this pathway and specialize later.

Compared to Cybersecurity, this pathway focuses on building software rather than breaking and protecting it. Security is covered as one module in Year 3 — enough for secure development practices, not enough for specialist security roles.

Computing is the right choice for students who want to become professional software developers — versatile, employable across a wide range of roles and industries, with strong practical skills in the tools and methodologies the industry actually uses.
